# Caso de Uso: Consultar Notas

## Objetivo

Permitir que o aluno consulte suas notas no sistema escolar.

## Ator Principal

- Aluno

## Pré-condições

- Usuário cadastrado
- Usuário autenticado

## Fluxo Principal

1. Aluno acessa o sistema.
2. Informa login e senha.
3. Sistema valida acesso.
4. Seleciona Consultar Notas.
5. Sistema exibe notas.

## Fluxo Alternativo

1. Visualizar boletim.

## Fluxo de Exceção

1. Login inválido.
2. Sistema informa erro.
